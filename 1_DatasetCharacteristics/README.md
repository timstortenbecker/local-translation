# Dataset Characteristics

**[Notebook](exploratory_data_analysis.ipynb)**

## Dataset Information

### Dataset Source
- **Dataset Link:** [manythings.org](https://www.manythings.org/anki/)


### Dataset Characteristics
- **Number of Observations:** 197147 sentences in english and portuguese


### Target Variable/Label
- **Label Name:** Portuguese
- **Label Type:** Translation
- **Label Description:** Give an english sentence predict the portuguese translation



## Exploratory Data Analysis

The exploratory data analysis is conducted in the [exploratory_data_analysis.ipynb](exploratory_data_analysis.ipynb) notebook, which includes:

- Data loading and initial inspection
- Statistical summaries and distributions
- Missing value analysis
- Feature correlation analysis
- Data visualization and insights
- Data quality assessment
